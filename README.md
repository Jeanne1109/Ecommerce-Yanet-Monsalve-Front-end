# Portafolio: Módulo 2 – Ecommerce (Princess Fedora)

## Descripción
Este proyecto implementa la primera versión (MVP) del frontend para "Princess Fedora", una boutique de ropa gótica y underground para mascotas. El proyecto cumple con los requisitos del Módulo 2 del Bootcamp "DESARROLLO DE APLICACIONES FULL STACK PYTHON TRAINEE" demostrando el uso de HTML5 semántico, diseño responsivo con Bootstrap, y manipulación del DOM con JavaScript para el contador de carrito.

## Características Clave
* **Estética:** Diseño gótico victoriano con paleta de colores oscuro y acentos rubí.
* **Responsive:** Layout optimizado para móvil y escritorio usando Bootstrap Grid.
* **Interactividad:** Contador de carrito funcional en la Navbar usando JavaScript y LocalStorage.
* **Funcionalidad Extra:** Carrusel de imágenes funcional en la página de detalle.
